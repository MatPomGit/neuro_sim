Symulator mezoskopowy procesów poznawczych mózgu

Wersja statyczna dla GitHub Pages. Obliczenia wykonuje Python uruchamiany w przeglądarce przez Pyodide. Interfejs odzwierciedla bieżące możliwości GUI wersji desktopowej: automatyczny dobór dt, scenariusze, tryb batch z analizą wrażliwości, odczyt behawioralny, wykresy aktywności/EEG/regionów, zapis/wczytanie konfiguracji JSON, eksport CSV oraz raport PDF z wykresami.
